# Product Category Classifier

Ovaj projekat razvija model mašinskog učenja koji automatski predviđa
kategoriju proizvoda na osnovu njegovog naziva.

## Struktura projekta

analysis_and_model_training.ipynb — kompletna EDA analiza, obrada podataka i treniranje modela

train_model.py — skripta za treniranje i čuvanje modela

predict_category.py — interaktivna skripta za ručno testiranje predikcija

product_category_model.pkl — trenirani model

products.csv — skup podataka
